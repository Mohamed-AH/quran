## ğŸ”§ Troubleshooting

### Can't Sign In

**Problem:** OAuth redirect not working

**Solutions:**
- Check your internet connection
- Try a different browser (Chrome recommended)
- Clear browser cache and cookies
- Make sure you're using the correct Google/GitHub account
- Try signing in again after 5 minutes

### Data Not Saving

**Problem:** Changes don't persist

**Solutions:**
- Check internet connection (cloud sync required)
- Look for error messages (red notifications)
- Refresh the page and try again
- Sign out and sign back in
- Contact support if issue persists

### Dashboard Not Updating

**Problem:** Statistics show old data

**Solutions:**
- Refresh the browser page (F5 or Cmd+R)
- Sign out and sign back in
- Clear browser cache
- Try a different browser

### Language Not Switching

**Problem:** Text stays in one language

**Solutions:**
- Click the language toggle again
- Refresh the page
- Clear browser cache
- Check browser console for errors

### Page Format Errors

**Problem:** "Invalid page format" error

**Solutions:**
- Use correct format: `1-5` or `1, 3, 5`
- Don't use spaces between numbers (or use commas)
- Don't exceed page 604
- Check for Arabic/English number confusion
- Examples: `1-10` âœ…  `1 - 10` âŒ  `1 10` âŒ

### PWA Installation Not Showing

**Problem:** Can't install as app

**Solutions:**
- **iOS:** Must use Safari browser
- **Android:** Use Chrome browser
- **Desktop:** Use Chrome or Edge
- Make sure you're on HTTPS connection
- Try visiting site multiple times
- Check browser settings allow installations

## ğŸ’¡ Tips for Success

### Memorization Best Practices

1. **Consistency Over Quantity**
   - Better to memorize 1 page daily than 7 pages once a week
   - Use streak tracking for motivation
   - Set realistic daily goals

2. **Regular Review**
   - Review previous pages before new memorization
   - Use the review pages field every session
   - Track which pages need more review

3. **Quality Ratings**
   - Be honest with your ratings
   - 5 stars means you can recite perfectly
   - Lower ratings indicate need for more review

4. **Notes are Valuable**
   - Document difficult verses
   - Note connections between ayahs
   - Record your achievements
   - Plan tomorrow's session

### Using Hafiz Effectively

1. **Daily Logging**
   - Log immediately after your session
   - Don't wait until end of day
   - Include both new and review pages
   - Add meaningful notes

2. **Juz Management**
   - Update Juz status regularly
   - Set start dates when beginning a Juz
   - Mark completion dates for milestones
   - Add Juz-specific notes

3. **Progress Tracking**
   - Check statistics weekly
   - Celebrate milestones (every Juz completed!)
   - Review history to see patterns
   - Adjust schedule based on progress

4. **Multi-Device Usage**
   - Sign in on all your devices
   - Data syncs automatically
   - Use phone for quick logs
   - Use tablet/computer for reviews

### Memorization Schedule Ideas

**Beginner Schedule:**
- New: 1 page per day
- Review: Previous 5 pages
- Rest: 1 day per week

**Intermediate Schedule:**
- New: 2 pages per day
- Review: Previous 10 pages
- Rest: 1 day per week

**Advanced Schedule:**
- New: 3-4 pages per day
- Review: Previous 20 pages (one Juz)
- Continuous with breaks for completed Juz

**Customize Your Own:**
- Listen to your capacity
- Adjust based on difficulty
- Some Juz are harder than others
- Quality over speed
